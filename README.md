# Django Sticky Notes Application

This repository contains my Django Sticky Notes project for:

- Task 16: Sticky Notes Application – Part 1
- Task 17: Sticky Notes Application – Part 2

## Features
- Create, view, update, and delete notes
- Django admin integration
- Bootstrap-styled templates
- Automated tests for models and views

## How to run
```bash
python -m venv .venv
.venv\Scripts\activate
pip install django
python manage.py migrate
python manage.py runserver
